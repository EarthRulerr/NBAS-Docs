Notepad Basic

Notepad Basic (NBAS) is a console based programming language based off of Shell and BASH. The appearance and usage is very basic, hence the name.

Installation

Windows / Mac / Linux

Fork and clone the github repository.

git clone https://github.com/EarthRulerr/NBAS

Documentation

Notepad Basic

Tags

PRNT string ; - Printing strings to console.
CMT string - Comments.
GET INPUT - Next string is recorded as INPUTVAL.
MGET INPUT - Multiline version of GET INPUT.
INPUTVAL - Prints INPUTVAL.
ADD # , # - Addition.
SUB # , # - Subtraction.
MUL # , # - Multiplication.
DIV # , # - Division.
POW # , # - Raised to the power.
WAIT # - Wait # seconds.

Console Reporters

$CPU - Reports CPU name.
$CPUS - Reports number of CPUs.
$CPUM - Reports CPU memory.
$CPU_ARCH - Reports CPU architecture.
$CPU_SPD - Reports CPU speed.
$CPU_FREQ - Reports CPU frequency.
$OS - Reports the operating system.
$TIME - Reports server time.
$IP - Reports IP address.
$USER - Reports host name.
$LOC - Reports location.
$LINE - Reports line.
$SYST - Reports system data.

Other

%20 - Prints a space.
CLR - Clears console.
PKGHELP - Help for packages.
HELP - Tag guide. GUIDES - Opens all guides and help.

Packages

(pkg.reset) - Resets settings to default.
(pkg.custom) - Installs Custom package, allows for console customization.
PKGHELP and GUIDES for help on packages.
